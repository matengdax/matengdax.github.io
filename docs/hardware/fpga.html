FPGA¶

可编程逻辑单元¶

可以通过编程变成你想要的电路功能

指的是 FPGA（现场可编程门阵列，Field Programmable Gate Array）芯片内部大量的小型数字电路模块

它们本身能实现与门、或门、非门等基本逻辑功能

但具体做什么功能，可以通过“编程”来决定

比如，你可以让某些逻辑单元变成加法器、计数器、触发器等

可编程互连资源¶

可以通过编程决定不同逻辑单元之间怎么连线

指的是 FPGA 芯片内部大量的“连线通道”和“电子开关”

这些资源不是像普通芯片那样出厂时就固定谁和谁连接

而是可以通过后天编程（配置）来决定哪些逻辑单元之间相连，信号怎么流动

FPGA为什么可以改变电路¶

传统ASIC芯片（专用集成电路）

  • Application Specific Integrated Circuit

  • 制造方式：通过光刻、蚀刻等工艺把晶体管、电路的连接“固化”在硅片上，电路一旦生产出来就不能更改。

  • 本质：硬件结构是固定的，不能变。

FPGA芯片（现场可编程门阵列）

  • Field Programmable Gate Array

  • 制造方式：FPGA同样通过光刻等方式制造，但制造出来的是大量的“可编程逻辑单元”和“可编程互连资源”。

  • 本质：这些逻辑单元和互连线在出厂时是“空白”的，内部的连接关系还没有确定。

  • 特点：你买到手的FPGA，内部的硬件连接是可以后期通过“配置”来改变的。但物理结构（逻辑单元、开关矩阵等）是早已“光刻”好的

FPGA内部结构

  • 有大量的逻辑块（如LUT、寄存器等）和可编程交换矩阵。

  • 还有一堆配置开关，这些开关决定了各逻辑块和互连线的连接方式

配置过程

  • 用户通过编程工具（比如烧录.bit文件）把自己的电路设计（逻辑连接关系）写进FPGA的配置存储器（如SRAM）。

  • 配置存储器的内容决定了内部的“开关”是开还是关，哪些逻辑块和哪些连线连接在一起。

  • 这就等于“后天”用电子开关把事先设计好的“电路模块”按你的意愿连起来了。

改变电路的原理

  • 不是重新“蚀刻”硅片，而是“动态改变”内部的电子开关状态。

  • 就像一个大号的乐高积木，积木本身形状是固定的，但你可以随时重新拼装出不同的模型。

配置的可逆性

  • 多数FPGA（如基于SRAM的）断电后配置会消失，可以多次重新配置，实现多次“改变电路”。
